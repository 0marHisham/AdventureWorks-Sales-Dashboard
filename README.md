# AdventureWorks Sales & Customer Analytics Dashboard

## Overview
**AdventureWorks** is a comprehensive **Power BI dashboard** built using the **AdventureWorks dataset**.  
It provides interactive insights into **sales, customers, products, and regional performance**, allowing stakeholders to make faster and smarter business decisions.

---

## Key Features
- **Overview KPIs:** Total Sales, Total Profit, Profit Margin %, Total Orders.  
- **Customer Insights:** Active Customers, Avg Spend per Customer, Customer Purchase Rate %.  
- **Product Performance:** Top Products, Total Units Sold, Avg Profit per Product, Product Return Rate %.  
- **Sales Analysis:** Total Sales trends, YoY Growth %, Top Region Sales, Average Order Value.  
- **Advanced Analytics:** Decomposition Tree to explore sales drivers (Region → Product Category → Product Name).  
- **Interactive Filters & Slicers:** Year/Month, Region, Product Category, Product Name.  

---

## Tools & Technologies
- **Power BI Desktop** for creating interactive dashboards  
- **DAX** for dynamic measures and calculations  
- **AdventureWorks dataset**
